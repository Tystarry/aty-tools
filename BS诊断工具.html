🧩 BS诊断工具 v2.0

ABC 流程 blendShape 诊断 — 输入组名 → 生成脚本 → Maya 运行 → 粘贴 JSON → 导出 PDF

📋 第一步：生成 Maya 脚本

填写组名后点复制，粘贴到 Maya Script Editor (Python) 运行

📊 第二步：粘贴诊断结果

将 Maya 输出的 JSON 粘贴到下方

📖 使用说明

1. 填写 Mod 组名和 ABC 组名

2. 点「复制脚本」→ 粘贴到 Maya Script Editor (Python) → 运行

3. 复制 Maya 输出的 JSON（包含在 === COPY JSON BELOW === 之后）

4. 粘贴到左侧文本区 → 点「生成清单」

5. 点「导出 PDF」发给其他环节

诊断项目

✅ 顶点数是否一致

🧊 Transform 是否冻结（位移/旋转/缩放是否归零）

🏗 层级结构是否一致（mod 和 ABC 的相对路径）

⚠ mod 独有 / ABC 独有的 mesh

注意

Mod 和 ABC 组必须同时存在于场景中

脚本扫描组下所有子 mesh，自动匹配同名比对

不含任何第三方依赖，纯 Maya Python
